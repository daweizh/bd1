3.1 基本环境准备

主要完善Hadoop需要的Ubuntu环境。

vim的使用方法：

  1. vim启动后一般会停留在“浏览”状态，此时只能看，不能改
  2. 在浏览状态按i键进入编辑状态，屏幕下端出现---INSERT---提示信息
  3. 退出编辑状态按Esc键
  4. 在浏览状态按:键，就是shift+;组合键，进入命令状态
  5. 在命令状态键入wq两个字母表示存盘并退出
  6. 在命令状态键入q!表示强制退出

3.1.1 基础准备

  1. 系统查新操作
    $ sudo apt-get update   
$ sudo apt-get upgrade
  2. 安装或更新ssh客户端程序。SSH(Secure Shell)安全外壳协议。
    $ sudo apt-get install ssh 
[sudo] password for jack:
... ...
Do you want to continue? [Y/n]
... ...
  3. rsync是linux系统下的数据镜像备份工具。Remote Sync可以远程同步，支持本地复制，或者与其他SSH、rsync主机同步。
    $ sudo apt-get install rsync
  4. wget是一个从网络上自动下载文件的自由工具，支持通过HTTP、HTTPS、FTP三个最常见的TCP/IP协议下载，并可以使用HTTP代理。"wget"这个名称来源于“World Wide Web”与“get”的结合。所谓自动下载是指wget可以在用户退出系统之后在后台继续执行，直到下载任务完成。
    $ sudo apt-get install wget
  5. tree 生成目录树
    $ sudo apt-get install tree
  6. vim编辑器
    $ sudo apt-get install vim

3.1.2 bd结构规划

我们将规划部署的内容命名为bd，其目录结构如下：

/usr/bd
├── dfs
│   ├── data
│   └── name
├── hadoop-2.7.5      注：软件安装时创建
├── jdk1.8.0_202      注：软件安装时创建       
├── logs
└── tmp

创建上述目录结构：

  1. 创建目录
    $ sudo mkdir /usr/bd    
$ cd /usr/bd    
$ sudo mkdir dfs logs tmp dfs/name dfs/data
  2. 修改权限
    $ sudo chown -R jack:jack dfs logs tmp
  3. 查看权限
    $ ls -ls
4 drwxr-xr-x  4 jack  jack                 4096 Mar 19 08:26 dfs
4 drwxr-xr-x  2 jack  jack                 4096 Mar 19 08:27 logs
4 drwxr-xr-x  2 jack  jack                 4096 Mar 19 08:27 tmp
$ ls -ls dfs
4 drwxr-xr-x 2 jack jack 4096 Mar 19 08:26 data
4 drwxr-xr-x 2 jack jack 4096 Mar 19 08:26 name

3.2 准备Java

  1. 进入/usr/bd文件夹
    $ cd /usr/bd
  2. 下载Java
    $ sudo wget http://210.47.218.146:8080/repos/courses/100509/150000004/tool/2019/jdk-8u202-linux-x64.tar.gz
jdk-8u202-linux-x64 100%[===================>] 185.05M  7.84MB/s    in 23s
  3. 解压Java
    $ sudo tar -zxvf jdk-8u202-linux-x64.tar.gz
  4. 配置Java
    $ sudo vim /etc/profile

if [ -d /etc/profile.d ]; then
  for i in /etc/profile.d/*.sh; do
    if [ -r $i ]; then
      . $i
    fi
  done
  unset i
fi
export JAVA_HOME=/usr/bd/jdk1.8.0_202
export PATH=$PATH:$JAVA_HOME/bin
  5. 执行/etf/profile使配置生效
    $ source /etc/profile
  6. 检查系统环境变量
    $ env
... ...
PATH=... ...:/usr/bd/jdk1.8.0_202/bin
... ...
JAVA_HOME=/usr/bd/jdk1.8.0_202
... ...
  7. 验证Java配置
    $ java -version
  java version "1.8.0_202"
  Java(TM) SE Runtime Environment (build 1.8.0_202-b08)
  Java HotSpot(TM) 64-Bit Server VM (build 25.202-b08, mixed mode)

3.3 准备Hadoop

  1. 进入/usr/bd文件夹
    $ cd /usr/bd
  2. 下载hadoop
    $ sudo wget http://210.47.218.146:8080/repos/courses/100509/150000004/tool/2019/hadoop-3.1.2.tar.gz  
hadoop-3.1.2.tar.gz 100%[===================>] 317.03M  7.82MB/s    in 40s
  3. 解压HADOOP
    $  sudo tar -zxvf hadoop-3.1.2.tar.gz
  4. 配置Path
    $ sudo vim /etc/profile

export JAVA_HOME=/usr/bd/jdk1.8.0_202
export PATH=$PATH:$JAVA_HOME/bin

export HADOOP_HOME=/usr/bd/hadoop-3.1.2
export PATH=$PATH:$HADOOP_HOME/bin:$HADOOP_HOME/sbin
  5. 执行/etf/profile使配置生效
    $ source /etc/profile
  6. 检查系统环境变量
